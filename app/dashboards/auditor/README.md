# Auditor Dashboard

Auditor/verifier dashboard for report verification and compliance checking.

## Features

- Report review and verification
- Compliance checking
- Audit trail access
- Verification workflows
- Blockchain record verification

## Routes

- `/auditor` - Auditor dashboard
- `/auditor/reports` - Report review
- `/auditor/verifications` - Verification management
- `/auditor/audit-trail` - Audit trail access
