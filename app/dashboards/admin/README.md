# Admin Dashboard

Administrator dashboard for system management and oversight.

## Features

- User management
- System configuration
- Emission factor management
- Report verification
- Analytics and insights

## Routes

- `/admin` - Admin dashboard
- `/admin/users` - User management
- `/admin/emission-factors` - Emission factor management
- `/admin/reports` - Report oversight
- `/admin/settings` - System settings
