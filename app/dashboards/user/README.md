# User Dashboard

Standard user dashboard for carbon accounting and reporting.

## Features

- Activity management
- Emission calculations
- Report generation
- Data visualization
- CBAM compliance (if applicable)

## Routes

- `/dashboard` - Main dashboard
- `/dashboard/activities` - Activity management
- `/dashboard/reports` - Report management
- `/dashboard/settings` - User settings
